# ZhiyiSerif ( 致一宋体 / 致一宋體 / 致一明朝 / 치일명조 ) 

![後之覧者，亦將有感於斯文](https://github.com/RiyipuSipino/ZhiyiSerif/blob/main/%E6%95%88%E6%9E%9C%E5%9C%96.png)

## 說明

雖世殊事異，所以興懷，其**致一**也。——《蘭亭集序》

「致一宋體」是一套以TrueType格式封裝、依照Unicode編碼的OpenType字型，採用「**懷有傳承字形向往的自訂新字形**」，即以傳承和慣用字形爲主體、多就少改，兼以傳承字形筆法爲基礎，參考歷史異形、類推隸定規則，制訂新的字理部件，力使**字理直觀、優化、可分**（註：優先取用傳世字理和強勢初理）**，字形通行、美觀、可認、兼顧寫印**。

使用本字型時，須遵從「[IPA開放字型授權協議 1.0版](LICENSE.md)」之規定。

## 信念

### 避免隸古定，且盡可能區別字源、統一部件；既求字理直觀清晰，也求字形美觀通用

「致一宋體」本爲作者自創世界觀的字型，**初心即是區別字源、統一部件**。最初的字形有大量初文本字，雖合古意，但是「似篆非隸，若存若亡」，體系復雜，辨識度差，自己讀寫都慢，就更不要指望那個世界觀的角色們會用了。

縱觀漢字發展史，字理本不可知、還要後人考證者有之，初理本不可知、千年來已有重構新字理、棄去實在可惜者有之，本係同音、近音甚至同義假借（如「開」，本讀作{啓}，訓讀爲{闓}），「鳩佔雀巢」甚至「鳩佔鳩巢」者有之，即便身世清楚，直接把字源堆到一起的「古字」大眾根本不識（如「⿱爫象」，今作「爲」）者亦有之。

究其原因，今人所書雖去古漸遠，然筆畫增損之間，實有潛藏理路可循。一曰**隸今定法則**，二曰**理據重構**。隸今定法則——即「現代隸定規律」——不是一對一的，而是一對多、多對一的。例如「出」「大」，作部件時多簡寫爲「士」「土」——字理上的「出」「大」不一定寫成「出」「大」，字形上的「士」「土」也未必就是「士」「土」。重構理據更是漢字傳承的生命力所在。例如「幸」字，本從犬、從倒矢，理據不明，今則從夭從屰（逆），理據昭然。

**傳承字形是隸今定法則的結晶**，經宋明以來歷代文字學者整理，已具備高度的一致性和區分性。不過，隨著考古學和文字學的進一步發展，**以前的某些字形已經不再是最優解**。例如，出土簡帛的聲部件，左上角從士形、從石，若取書法形體，理據重構從十形、從石，則比印刷體異變後的純象形理據更直觀；現代文字學已經證明，「益」與（「溢」之右）並不相同，而傳承字形雖有兩種不同寫法（「益」與「益」），卻受當時的學術所限，相互混作，並未特意分開。

正式版謹守隸今定法則、尊重理據重構，並經多番考量，最終決定：以傳承字形爲基礎，以約定俗成爲準則，多就少改、應改即改，讓字源和重構理據公平競賽，**既求字理直觀清晰，也求字形美觀通用**。不獨是爲了自創世界觀的角色，亦是爲了做一份文字學答卷，請天下有緣之士批評指正。
